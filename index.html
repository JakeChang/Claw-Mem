Claude Code
工作記憶助手

Claw-Mem 自動監控 ~/.claude/projects 的 JSONL 紀錄，透過 Google Gemini 生成結構化摘要，讓你每天無縫延續開發上下文。

下載 DMG 查看原始碼

需要 macOS 15.0 或以上 · Apple Silicon & Intel

功能特色

記憶你的每一次工作

即時監控 JSONL 紀錄

背景持續偵測 ~/.claude/projects 目錄，有新工作紀錄即自動解析入庫，無需手動觸發。

AI 生成結構化摘要

透過 Google Gemini 串流生成每日工作摘要，涵蓋完成事項、修改檔案清單、遇到的問題與解法，以及未完成的待辦事項。

專案總摘要

從所有歷史摘要歸納整個專案的全貌：技術棧、架構設計、里程碑與目前狀態。

一鍵複製給 Claude

格式化摘要為 prompt，貼到新對話即可讓 Claude 立刻掌握上下文，無縫延續開發。

日期篩選

快捷選擇 7 / 30 / 90 天，或自訂日期範圍，快速聚焦特定開發時期的紀錄。

跨裝置同步

透過 Dropbox 或 iCloud 資料夾同步摘要與備註，在多台 Mac 之間保持工作記憶一致。

Menu Bar 常駐顯示

右上角選單列即時顯示今日活躍專案數與訊息紀錄數，讓你隨時掌握今天的工作量，點擊即可展開完整的對話瀏覽器。

快速設定

取得 Gemini API Key

1

前往 Google AI Studio

開啟 aistudio.google.com，使用 Google 帳號登入後，點擊「Get API key」→「Create API key」。

2

貼入 App 設定

開啟 Claw-Mem → 右上角設定，將 API Key 貼入對應欄位後按儲存。預設使用 Gemini Flash Lite 模型，也可自行切換。

3

開始生成摘要

App 會自動偵測現有的 JSONL 紀錄並開始解析。對任何一天的工作紀錄點擊「生成摘要」，Gemini 即以串流方式輸出結構化內容。

安裝指南

兩步驟開始使用

1

下載 DMG

前往 Releases 下載最新版本，開啟 DMG 並將 Claw-Mem.app 拖入 Applications 資料夾。

2

移除隔離屬性 （必要步驟）

由於 App 未經 Apple 公證，首次開啟前請在終端機執行以下指令，否則 macOS 會阻擋啟動：

xattr -cr /Applications/Claw-Mem.app
